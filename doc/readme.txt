------IDE SETTING------
-----------------------
org.androidannotations
-----------------------

1. open the setting window of this module
2. In modules view click the "Android" to show the right window "Facet Android"
3. switch to "compiler" ,select the radio "Generated by IDE" ,and specify the value ${your project context path}/target/generated-sources/r
4. AIDL files specify the value ${your project context path}/target/generated-sources
5. add the dependencies to the classpath from maven library
6. remove the context root of default "gen"
7. Running&Debug: add clean and compiler before make
8. setting maven profile refer to settings.xml within folder doc
9. Open the Preference of the Intellij IDEA,find compiler--Annotation Processor,there should be a "maven default annotation processors profile" at the middle of the panel.
   Enable annotation processing and then click ok.
10.No.9 will output something against the file :.idea/compiler.xml.You will find the profile there.

------DATABASE SETTING------
----------------------------
1. doc:http://ormlite.com
2. You need generate the meta data for database by using DatabaseConfigUtil after updating table entities.
   It will  output a ormlite_config.txt in the folder : raw
3. There are two different ways for usage of database.One is creating database manually(Refer to DBActivity),the other one can manipulate the database by extends class OrmLiteBaseActivity.
   com.join.android.app.common.db.HelloAndroid will be a good example for the case.
